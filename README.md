☕ Cafe Sales Analysis Dashboard (Excel)
📌 Project Overview
This project is an interactive Cafe Sales Analysis Dashboard built in Microsoft Excel. 
It provides insights into sales performance using Pivot Tables, Pivot Charts, Slicers, and
KPIs, helping users analyze sales data efficiently.

🎯 Objectives
Analyze overall cafe sales performance.

Track revenue, transactions, and quantity sold.

Compare sales by product, category, location, and time.

Create an interactive dashboard for quick decision-making.

🛠 Tools Used
Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

KPI Cards

Data Cleaning

Conditional Formatting

📊 Dashboard Features
Total Revenue

Total Transactions

Total Quantity Sold

Top 10 Products by Quantity

Sales by Location

Hourly Sales Trend

Product Category Distribution

Product Type Distribution

Size-wise Sales Analysis

Interactive Filters (Day & Month)

📈 Key Insights
Identified the highest-selling products.

Compared sales across different store locations.

Analyzed hourly sales patterns.

Examined product category and size distribution.

Enabled dynamic filtering for faster analysis.
